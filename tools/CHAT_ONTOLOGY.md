# 💬 상담 키워드 온톨로지 — 7축 조합 체계

상담의 모든 경우의 수를 **7개 키워드 축의 조합**으로 다룬다. 축마다 키워드 사전(정규식)이 있고,
각 축은 검색(후보 선정)과 답변(LLM 프레이밍)에 **독립적으로 기여**하므로, 축 값들의 조합
(5×9×5×7×12×4×3 ≈ 확장 조합 수십만)이 전부 일관된 상담으로 합성된다. 축을 열거하는 방식이라
새 키워드는 사전에 한 줄 추가로 전체 조합에 반영된다.

구현 위치: `tools/app_base.html`의 `CHAT_GRADE/SKILL/LEVEL/WEAK/GOAL/TIMING/MODE` + `parseChat` + `chatRetrieve` + `chatSend`(프로필 전달).

## 축 정의

| 축 | 값(canonical) | 키워드 예 | 검색 효과 | 답변 연결 |
|---|---|---|---|---|
| ① 학년 | 유아/예비초·초등·중등·고등·성인 | 유치원, 초3, 중학생, 고1, 수능, 토익 | **하드필터**(그 학년/전체만) + 기본수준 | 없으면 **되묻기 1순위** |
| ② 영역 | 문법·독해·어휘·듣기·말하기·쓰기·파닉스·구문·모의/기출 | 어법, 리딩, 단어, 회화, 영작 | 풀 필터 + 25점 | 없으면 **되묻기 2순위**, 없+모호 → 영역 다양화 |
| ③ 수준 | 1~5 연속 | 영포자(1.5), 기초(2), 기본(3), 실전(4.5) / N등급 / NN점 / "더 쉬운 걸로"(조정) | 레벨 거리 감점 ×6 | 헤더에 기초/기본/실전 |
| ④ 약점 | 서술형·빈칸추론·어법·구문독해·시간관리·어휘력·말하기 | "서술형에서 틀려", "시간이 부족" | 보완 영역 +14 | LLM이 ①진단→②교재→③순서 구조로 답 |
| ⑤ 목표 | **교재 goals 태그 12종과 동일 어휘** — 내신 대비·수능·모의고사·시험영어(토익·토플)·영어 첫걸음·학교영어 따라가기·독해 늘리기·문법 잡기·어휘 늘리기·듣기·말하기·회화·쓰기·서술형·원서 읽기 | 내신, 기말고사, 토플, 알파벳부터 (명시 없으면 영역에서 유도) | **b.goals 태그 매칭 +16** | 헤더 🎯, LLM 프로필 |
| ⑥ 시기 | **교재 timing 태그 4종과 동일 어휘** — 방학 단기집중·시험 직전·학기중 꾸준히·기초 입문 | 방학, 벼락치기, D-30, 꾸준히 (기초 수준→기초 입문 유도) | **b.timing 태그 매칭 +12** | 헤더 🗓, LLM 프로필 |
| ⑦ 상황 | 독학·엄마표·학원 병행 | "혼자 공부", "엄마표", "과외 병행" | (검색 중립) | LLM 프로필 → 해설 상세/부모 가이드/보조 교재 프레이밍 |

+ 보조 신호: **원서**(wantForeign → foreign/영문제목 +30), **한자 방어**(3중), **시효성**(구판 시험서 -35).

## 합성 규칙 (경우의 수 → 하나의 답)

1. **파싱**: 질문에서 7축 값 추출 → 빈 축은 ①직전 대화 승계 ②유도 규칙(영역→목표, 기초수준→기초입문 시기, 학년→기본수준) ③핵심 축(①②)이 그래도 비면 **되묻기(칩)**.
2. **검색**: 하드필터(학년·영역·절판·시효) 후 축별 가중치 합산 → 시리즈 dedup 6권.
3. **답변**: `[파악된 정보: 학년=고등, 목표=수능·모의고사, 약점=서술형, 수준=Lv4.5, 시기=방학 단기집중, 상황=독학]`을 질문에 붙여 LLM 전달 → 진단→교재→순서 구조로 조합 맞춤 답변. 검색 폴백 템플릿 헤더에도 동일 축 표기.

## 조합 예시

| 질문 | 파싱된 조합 | 답변 방향 |
|---|---|---|
| "고1 2등급, 서술형 약해, 방학 두 달, 혼자 공부" | 고등×쓰기·문법×Lv4.5×서술형×내신대비×방학단기×독학 | 어법→서술형 실전 순서 + 해설 상세 교재 |
| "유치원생 파닉스 시작" | 유아×파닉스×Lv1.5×영어첫걸음×기초입문 | 첫걸음 단계 파닉스 |
| "직장인인데 토익 700 넘기고 싶어" | 성인×모의/기출×시험영어(토익) | 성인 시험영어 트랙 |
| "쉬운 원서 읽고 싶어" | 원서희망×Lv2×원서읽기 | 실원서(영문제목) 우선 |
| "문법" (정보 부족) | 영역만 → 학년 되묻기 → 칩 답변과 합성 | 슬롯필링 |

## 유지보수

- 새 키워드: 해당 축 사전에 정규식 추가만 하면 전 조합에 반영.
- 새 축: 사전 + parseChat 추출 + (검색 효과 or LLM 프로필) 한 줄씩.
- 전전두엽(주 1회)이 실로그에서 미매칭 표현을 발견해 `brain_insights.md`에 패턴 후보를 제안 → 여기 사전에 반영하는 루프.
